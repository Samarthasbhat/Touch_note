# Touch_note :bookmark:
Notes application for android


## _Features_

:page_facing_up:**Add note**
 
 :date: **Date and :clock1: time will be shown**
 
 :pushpin:**Pin/Unpin**
 
 :mag:**Search**

 :wastebasket:**Delete**
 
### With ascending order it will store note

**Icons used from [icons8](https://icons8.com/)**
